Previous efforts by Sergi Granell (xerpi) on Linux for the 3DS is now continued at Linux for 3DS. I discovered them on this Reddit post.

Nick Desaulniers stream on Twitch1 has showcased the current progress and future plans for Linux for 3DS.

I made my share of helping the team out by providing a Dockerfile for settting up a build environment. #15, #8

I also created automated builds with Github Actions so that every time a push is made to their repositories, Github Actions runs the build and automatically uploads the binaries to Releases as development builds. #11, #20, #37

Everything is just amazing
